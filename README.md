# SecOps Portfolio

Portafolio técnico en construcción orientado a SecOps / SOC Analyst Jr.

Su objetivo es documentar prácticas de análisis de logs, detección, triage, runbooks y automatización ligera para construir evidencia defendible en seguridad defensiva operativa.

## Estructura

- logs-analysis/: análisis de eventos y logs
- detections/: reglas, lógica de detección y casos de uso
- cases/: investigaciones documentadas
- runbooks/: procedimientos de respuesta
- scripts/: automatización ligera
- notes/: notas técnicas y glosarios
- assets/: capturas, diagramas y evidencia visual
